How I managed to build my own website within two weeks

26 Apr 2020

No, it’s not another fancy advertisment for Wix.com, or a similar platform, we’re talking here about real coding, hours of fixing bugs, website deployment hassles and headaches.

My coding background

To be honest no, it’s not my first coding experiance, I’ve +1.5yr in real-world Android development field, with 5 freelancing projects, and an award for the best HMS app by Huawei. but we still talking here about 0 experiance in web development/design fields, How did I manage to do it?

Computer science education cannot make anybody an expert programmer any more than studying brushes and pigment can make somebody an expert painter. Eric Raymond

One of the most impactful qoutes on my programming career, learning coding isn’t about just sitting in front of a tutorial and then re-write the same code you have seen, it’s about just learning syntax, and then start working on your own idea, or that was what I did. They say to make a website you need to learn HTML, CSS, Javascript, bootstrap and then if you wanna build a blog you need to learn a back-end framework or a language like PHP or Node.js, but I didn’t learn about anything. All what I did was watching a HTML course and I was dying, it was so boring, then started the CSS course and after 30 minutes I said “Hell yeah, I’d rather die than watching this shit”. I kept searching for another courses, videos, written tutorials and I realised that my only way to learn is by starting working on a real project, which’s my personal website. I hate that normal way of learning. And simply I started working on my beautiful website. When I needed to use Javascript (an important programming language in web design) in my website, I put the design I wanted in my head and then started searching about what I need, and because I was already familiar with programming language, I was writing in javascript without even wartching a single minute of a tutorial. When I needed to make a system for this blog I went to the easiest option for me which didn’t need any extra programming language. And I realised the truth of coding, it’s all about thinking like a computer, it’s all about using your brain, programming languages are just tools to implement ideas, the heavy work is on you not the tool.

what did I learn?

After finishing my website, I’m now familiar with web design world, I can’t make any kind of websites like a professional experianced developer, but if I wanted to, couple of month will be enough to learn the rest of web development stack, because while making this website I’ve already learned a lot, that’s what I like about “learning by projects” approach, it saved my time a lot. And the most important, experience, there’re things that courses won’t give you, this little tweaks and tricks that you will figure by yourself.

Is it the right way to learn web design?

Well yes but actually no, starting a professional career requiers real solid knowledge, what I did was just a personal challenge, I can’t consider myself now a “Web developer” and I can’t accept freelancing projects offers, I’m just a hobbyist, I wanted to make a website to share what I’m doing, but I didn’t want to learn how to make websites, that’s my point. So if you share the same my goal with my my approach will be perfect for you.

How can you do it?

Just watch an HTML course, it’s the most important, the backbone of whatever you will do, then just watch 30 minutes of CSS basics, like what I did and they was really enough for me, if you need to know how to do something else, you can search about it while working. and then complete a Javascript tutorial if you didn’t have any prior experiance with programming. Why I said you need to complete the whole javascript course? Because it’s completly diffrenent from HTML and CSS, it’ll be your real introdution for programming world. Did you finish the Javascript course? Congratulations, in less than 12 hours you’re able now to start your first professional website. Isn’t it easy like a piece of cake?
